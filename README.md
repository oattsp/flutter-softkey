# flutter_qtv

Queue TV Flutter application.

### Language
- dart

### Framework
- flutter

### Requirements
- tcp/ip socket client.
- show the waiting queue number of each group.
- show the holding queue number of each group.
- call queue by user.
- hold queue by user.
- end queue by user.
